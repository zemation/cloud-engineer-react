# A Cloud Engineer

A personal resume and portfolio site built with React — showcasing experience in cloud infrastructure, Linux administration, and full-stack development.

🌐 **[acloudengineer.com](https://acloudengineer.com)**

---

## Pages

- **Home** — Introduction and specialty overview
- **Resume** — Work history, certifications, and skills with interactive filtering
- **Projects** — Technical home lab and infrastructure projects
- **About** — Background, career timeline, and personal info

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, React Router, React Bootstrap |
| Styling | Bootstrap 5 |
| Icons | Font Awesome |
| Hosting | Self-hosted Kubernetes cluster |
| Monitoring | Prometheus, Grafana |
| CI/CD | Jenkins |

---

## Getting Started

### Prerequisites
- Node.js 18+
- npm

### Install

```bash
git clone https://github.com/zemation/cloud-engineer-react
cd cloud-engineer-react
npm install
```

### Run locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for production

```bash
npm run build
```

Output goes to the `build/` folder, ready to serve.
